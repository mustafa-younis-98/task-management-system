# Task Management System

A simple task management application built with Angular 19 and Bootstrap 5.

## Features

- Dashboard
- View Tasks
- Add Task
- Edit Task
- Delete Task
- Search Tasks
- Filter by Status
- Filter by Priority
- Task Details
- Loading Spinner
- Toast Notifications
- 404 Not Found Page
- Responsive Design

## Technologies

- Angular 19
- TypeScript
- Bootstrap 5
- RxJS
- Angular Router
- Reactive Forms
- HttpClient

## Installation

```bash
npm install
```

## Run the Project

```bash
ng serve
```

Open your browser and visit:

```text
http://localhost:4200
```
